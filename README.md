# Blackjack
Vue.js implementation of [RDeJonghe's](https://github.com/RDeJonghe) blackjack code 

**WORK IN PROGRESS**

Steps to install:
1. `git clone` onto local machine
2. `cd` into root directory
3. `npm install`
4. `npm run serve`
5. Navigate to `http://localhost:8080/` in a web browser
